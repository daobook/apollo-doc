定制¶

如果想要自定义网址缩写，只需要在 /conf.py 中修改 extlinks 变量：

# 缩短链接
extlinks = {
    'daobook': ('https://daobook.github.io', 'Dao Book'),
    'xinetzone': ('https://xinetzone.github.io', 'xinetzone'),
}

参见

详细内容可以参考：sphinx.ext.extlinks。
